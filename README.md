# Adaptive API Rate Limit & Abuse Detection Gateway

This project implements an adaptive API gateway that protects backend services from abuse and high traffic using intelligent rate limiting and risk scoring.

## Features
- Token Bucket Rate Limiting
- Adaptive user request limits
- API abuse detection
- Risk scoring system
- Request analytics

## Tech Stack
- Java
- Spring Boot
- Redis
- MySQL
